# Basic Network Sniffer

A simple Python-based network sniffer built with [Scapy](https://scapy.net/).  
This tool captures live network packets, analyzes their structure, and displays useful information such as source/destination IPs, protocols, ports, and payloads.

---

## 📌 Purpose
- Learn how data flows through a network.
- Understand the basics of protocols (IP, TCP, UDP).
- Practice packet capturing and analysis using Python.
